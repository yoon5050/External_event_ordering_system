🔧 최초 1회 설정이 필요합니다

이 플랫폼은 Firebase(구글 무료 서비스)로 로그인과 데이터 공유를 처리합니다.
파일 안의 firebaseConfig 부분에 프로젝트 정보를 붙여넣어야 작동해요.

1. console.firebase.google.com → 프로젝트 만들기
2. Authentication → 시작하기 → Google 로그인 사용 설정
3. Firestore Database → 데이터베이스 만들기 (프로덕션 모드) → 규칙에 도메인 제한 규칙 붙여넣기
4. 프로젝트 설정(⚙️) → 일반 → 내 앱 → 웹 앱 추가 → firebaseConfig 값 복사
5. 이 HTML 파일을 메모장으로 열어 PASTE_YOUR_CONFIG 부분을 복사한 값으로 교체 후 재업로드
